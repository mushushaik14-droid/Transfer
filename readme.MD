# B. Tech Student Transfer Certificate (TC) Webpage

## 1. Description

The **B. Tech Student Transfer Certificate (TC) Webpage** is a simple and structured webpage developed using **HTML5**. It represents a college-issued Transfer Certificate containing student information, academic records, fee clearance details, and signature fields.

---

## 2. Vision / Purpose

The purpose of this project is to practice HTML document structure by designing a professional Transfer Certificate webpage.

---

## 3. Technologies Used

- HTML5

---

## 4. Features

- College header section
- College information
- Transfer Certificate title
- TC Number and Date of Issue
- Student Details section
- Academic Details section
- Fee Details section
- Signature section
